Davinci Observation Category Codes

Categories codes that classifies the observation for searching, sorting and display purposes when using Observation for Da Vinci project reporting

This code system http://hl7.org/fhir/us/davinci-deqm/CodeSystem/observation-category defines the following codes:

CodeDisplayDefinition
administrative AdministrativePertaining to *non-clinical* activities related to an organization's or business' administration and management. For example, medication reconciliation.
