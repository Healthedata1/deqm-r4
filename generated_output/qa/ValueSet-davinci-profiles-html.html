Da Vinci Profiles

Codes that represent the Da Vinci project set of profiles. The concept codes are equivalent to the StructureDefinition.id, the Display to the StructureDefinition. element and the defiinition are from the StructureDefinition.description.

This value set includes codes from the following code systems:

  • Include these codes as defined in http://hl7.org/fhir/us/davinci-deqm/CodeSystem/davinci-profiles
    CodeDisplay
    measurereport-mrpDa Vinci MeasureReport MRP ProfileThe Da Vinci MeasureReport MRP Profile is based on the Da Vinci MeasureReport Profile and constrains the `measure` element to medication reconciliation post-discharge and the `evaluatedResources` element to Task or Observation profiles.
    observation-mrpDa Vinci Observation MRP ProfileThe Da Vinci Observation MRP Profile is based upon the Da Vinci Observation Measure Profile and constrains the code(s) for representing the medication reconciliation post-discharge.
    task-mrpDa Vinci Task MRP ProfileThe Da Vinci Task MRP Profile is based upon the Da Vinci Task Measure Profile and constrains the code(s) for representing the medication reconciliation post-discharge.
    location-davinciDa Vinci Location ProfileThe Da Vinci Location Profile defines the constraints for representing the coverage id.
    coverage-davinciDa Vinci Coverage ProfileThe Da Vinci Coverage Profile defines the constraints for representing the coverage where the Medication Reconciliation records are kept.
    organization-davinciDa Vinci Organization ProfileThe Da Vinci Organization Profile defines the constraints for representing the organization whch is responsible for the Medication Reconciliation.
    patient-davinciDa Vinci Patient ProfileThe Da Vinci Patient Profile defines the constraints for representing a patient who is the subject of a Medication Reconciliation.
    practitioner-davinciDa Vinci Practitioner ProfileThe Da Vinci Practitioner Profile defines the constraints for representing the practitioner who is responsible for the Medication Reconciliation.
    practitionerrole-davinciDa Vinci PractitionerRole ProfileThe Da Vinci PractitionerRole Profile defines the constraints for representing the role which is responsible for the Medication Reconciliation.
